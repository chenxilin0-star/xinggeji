# 型格记 - 心理测试小程序

微信小程序云开发版心理测试应用，支持5种心理测试。

## 功能特性

- **5个心理测试**：MBTI人格测试、恋爱脑测试、性格动物测试、恋爱依恋类型、情绪压力自评
- **云函数支持**：用户登录、测试列表、测试详情、结果提交、历史记录
- **云数据库**：测试数据、用户记录存储
- **精美UI**：紫色渐变主题，圆角卡片设计，雷达图可视化

## 项目结构

```
型格记/
├── cloudfunctions/          # 云函数目录
│   ├── login/               # 用户登录
│   ├── getTestList/         # 获取测试列表
│   ├── getTestDetail/       # 获取测试详情
│   ├── submitTest/          # 提交测试
│   └── getMyRecords/        # 获取历史记录
├── miniprogram/             # 小程序目录
│   ├── pages/
│   │   ├── index/           # 首页（测试列表）
│   │   ├── test/            # 测试答题页
│   │   ├── result/          # 结果页（雷达图+维度分析）
│   │   ├── records/         # 历史记录
│   │   └── my/              # 个人中心
│   ├── utils/               # 工具函数
│   └── images/              # 图片资源
└── database/                # 数据库初始化脚本
```

## 快速开始

### 1. 创建云开发环境
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 开通云开发
3. 获取环境ID

### 2. 配置项目
1. 打开 `project.config.json`，修改 `appid` 为你的小程序AppID
2. 修改 `miniprogram/app.js` 中的 `env` 为你的云开发环境ID

### 3. 上传云函数
在微信开发者工具中，右键点击 `cloudfunctions` 目录，选择「上传并部署」

### 4. 初始化数据库
1. 在云开发控制台中创建集合：
   - `users` - 用户信息
   - `user_records` - 测试记录

## 测试说明

### MBTI人格测试
- 20题，4维度(E/I, S/N, T/F, J/P)
- 结果展示：四宫格可视化 + 雷达图

### 恋爱脑测试
- 15题，4维度(沉没成本/理想化/情绪依赖/失去理性)
- 雷达图展示各维度得分

### 性格动物测试
- 15题，4种动物(狮子/孔雀/考拉/猫头鹰)
- 雷达图展示动物性格占比

### 恋爱依恋类型
- 18题，2维度(焦虑/回避)
- 四象限结果：安全型/焦虑型/回避型/恐惧型

### 情绪压力自评
- 21题，3维度(抑郁/焦虑/压力)
- 雷达图展示心理健康指数
