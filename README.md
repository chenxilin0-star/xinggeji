# 型格记 - 心理测试小程序

微信小程序云开发版心理测试应用，支持5种心理测试。

## 功能特性

- **5个心理测试**：MBTI人格测试、恋爱脑测试、性格动物测试、恋爱依恋类型、情绪压力自评
- **云函数支持**：用户登录、测试列表、测试详情、结果提交、历史记录
- **云数据库**：测试数据、用户记录存储
- **精美UI**：紫色渐变主题，圆角卡片设计

## 项目结构

```

│   └── getMyRecords/        # 获取历史记录

│   │   ├── test/            # 测试页

│   │   ├── records/         # 记录页
```

## 快速开始

### 1. 创建云开发环境
1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 开通云开发
3. 获取环境ID

### 2. 配置项目
1. 打开 `project.config.json`，修改 `appid` 为你的小程序AppID
2. 修改 `miniprogram/app.js` 中的 `env` 为你的云开发环境ID

### 3. 上传云函数
在微信开发者工具中，右键点击 `cloudfunctions` 目录，选择「上传并部署」

### 4. 初始化数据库
1. 在云开发控制台中创建集合：
   - `users` - 用户信息
   - `user_records` - 测试记录

## 测试说明

### MBTI人格测试
- 20题，4维度(E/I, S/N, T/F, J/P)
- 计分规则：选项A计2分，选项B计0分

### 恋爱脑测试
- 15题，4维度(沉没成本/理想化/情绪依赖/失去理性)
- 计分规则：B选项+1分

### 性格动物测试
- 15题，4种动物(狮子/孔雀/考拉/猫头鹰)
- 计分规则：选项A=1分, B=3分, C=5分, D=7分

### 恋爱依恋类型
- 18题，2维度(焦虑/回避)
- 四象限结果：安全型/焦虑型/回避型/恐惧型

### 情绪压力自评
- 21题，3维度(抑郁/焦虑/压力)
- 反向计分题：6,8,9,11,13,15,17,19,20
